# ZeroLoss
今回のアドベントカレンダー
A：日付ロジック（Java） 
B：カレンダーUI（HTML/JS） 
C：CSS・画像・演出 
毎日最後に進捗共有（5分でOK）


Aの仕事は大きく3つです。
1️⃣ 今日の日付を取得する
2️⃣ 未来日付を押せなくする
3️⃣ 経過日数で画像を6段階切り替える
A担当の具体タスクまとめ
作業	完了条件
今日の日付取得	today取得成功
未来ボタン無効化	今日より後は押せない
経過日数計算	daysPassed取得
6段階分岐	stage決定
メッセージ表示	クリックで表示


B担当のゴール
1️⃣ 1〜25の日付ボタンを表示
2️⃣ カレンダーらしく配置
3️⃣ クリック時に日付を送信
4️⃣ メッセージ表示エリアを作る
B担当の具体タスク一覧
作業	完了条件
1〜25表示	ボタン25個並ぶ
グリッド配置	5×5で整列
クリック送信	dayがServletへ渡る
メッセージ表示	${message}表示成功
ハイライト	今日が色変わる



C担当のゴール
1️⃣ カレンダーを見やすくする
2️⃣ 成長画像6種類を作る
3️⃣ レベルアップ感を演出する
4️⃣ 全体デザインを統一する
C担当の具体タスク一覧
作業	完了条件
カレンダー配置	5×5で綺麗に表示
押せる/押せない色分け	未来はグレー
画像6種作成	imagesフォルダに保存
アニメーション追加	画像がふわっと出る
全体配色統一	テーマカラー決定
